Surveyer is an open-source tool I built for systematic literature reviews: 
It handles paper discovery, deduplication, relevance scoring, and PRISMA survey workflows.
It is specifically tailored to survey research papers since it can generate a very detailed PRISMA figure, a BibTex file with per-paper citation and an excel file containing all information about included and excluded papers.
